# Specification - Bổ sung dự án Motorbike Sales System

Track này thực hiện việc tích hợp dự án "Motorbike Sales & Management System" từ GitHub `tigcom/MotorbikeSalesSystem` vào danh mục dự án trong Portfolio.

## Mục tiêu
- Thêm dự án Motorbike Sales System vào `src/data/projects.js` ở vị trí số 4.
- Làm nổi bật khả năng quản lý đa chi nhánh và các tính năng nghiệp vụ đặc thù cho ứng dụng Desktop.
- Đảm bảo tính liên kết (Prev/Next) giữa các dự án không bị gián đoạn.

## Yêu cầu kỹ thuật
- **Slug:** `motorbike-sales-system`
- **Tech Stack:** Java Swing, JDBC, SQL Server, Apache POI.
- **Vị trí:** Số 04 (sau Sixdo E-commerce, trước Phong Vu Shop Clone).

## Nội dung chi tiết (Dự kiến)
- **Title:** Motorbike Sales & Management System
- **Subtitle:** Multi-branch Desktop Solution
- **Overview:** Hệ thống quản lý kinh doanh xe máy toàn diện dành cho các đại lý có quy mô lớn với nhiều chi nhánh, tích hợp từ khâu nhập hàng, bán hàng đến bảo hành.
- **Problem:** Đồng bộ hóa dữ liệu tồn kho giữa nhiều chi nhánh khác nhau, quản lý lịch bảo dưỡng định kỳ và tự động hóa báo cáo doanh thu.
- **Process:** Xây dựng giao diện Desktop trực quan với Java Swing, thiết kế database quan hệ chặt chẽ trên SQL Server và tối ưu hóa các tác vụ CRUD cho nghiệp vụ bán lẻ.
