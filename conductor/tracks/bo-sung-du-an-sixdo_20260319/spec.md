# Specification - Bổ sung dự án Sixdo E-commerce

Track này thực hiện việc tích hợp dự án "Sixdo E-commerce Platform" từ GitHub `tigcom/cloneSixdo` vào danh mục dự án trong Portfolio.

## Mục tiêu
- Thêm dự án Sixdo vào `src/data/projects.js` ở vị trí số 3.
- Cập nhật thông tin chi tiết về Tech Stack, Overview, Problem và Process dựa trên mã nguồn thực tế.
- Đảm bảo tính liên kết (Prev/Next) giữa các dự án không bị gián đoạn.

## Yêu cầu kỹ thuật
- **Slug:** `sixdo-ecommerce`
- **Tech Stack:** Java Spring Boot, Next.js, React, PostgreSQL, Flyway, Spring Security.
- **Vị trí:** Số 03 (sau Traveloka Clone, trước Phong Vu Shop Clone).

## Nội dung chi tiết (Dự kiến)
- **Title:** Sixdo E-commerce Platform
- **Subtitle:** Modern Fullstack Shopping Solution
- **Overview:** Nền tảng thương mại điện tử đồng bộ giữa Backend (Spring Boot) và Frontend (Next.js), tập trung vào hiệu năng và trải nghiệm người dùng mượt mà.
- **Problem:** Xử lý đồng bộ dữ liệu giữa hai nền tảng khác nhau, quản lý trạng thái giỏ hàng và bảo mật API với Spring Security.
- **Process:** Thiết kế API RESTful, triển khai database migration với Flyway, tích hợp React Server Components của Next.js cho frontend.
