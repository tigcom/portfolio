# Specification: Hoàn thiện danh mục dự án từ tài khoản tigcom

## 1. Mục tiêu
Bổ sung các dự án chất lượng cao từ tài khoản GitHub `tigcom` vào Portfolio để minh chứng cho khả năng làm việc với đa dạng Framework (Angular, Spring Boot) và các hệ thống nghiệp vụ phức tạp.

## 2. Các dự án sẽ được bổ sung
Dựa trên kết quả nghiên cứu, 3 dự án sau sẽ được đưa vào:

1. **Banking System (Full Stack):** Hệ thống Internet Banking hiện đại.
   - **Tech:** Angular 18, TypeScript, Java Spring Boot, MySQL.
   - **Key Features:** Quản lý tài khoản, chuyển tiền, lịch sử giao dịch, bảo mật tài chính.
2. **Motorbike Sales System:** Hệ thống quản lý cửa hàng xe máy.
   - **Tech:** Java, Spring Boot, Hibernate, MySQL.
   - **Key Features:** Quản lý kho, hóa đơn, khách hàng, báo cáo doanh thu.
3. **Youtube Clone:** Bản sao của nền tảng chia sẻ video Youtube.
   - **Tech:** Java, JSP/Servlet (hoặc Spring Boot tùy version), HTML/CSS.
   - **Key Features:** Xem video, tìm kiếm, danh mục nội dung.

## 3. Yêu cầu triển khai
- **Dữ liệu:** Cập nhật `src/data/projects.js` với đầy đủ các trường thông tin (Overview, Problem, Process, Results).
- **Mô phỏng UI:** Tạo UI Mockup cho trang Dashboard của **Banking System** bằng mã nguồn Vue 3 để làm ảnh minh họa.
- **GSAP:** Áp dụng các hiệu ứng animation đồng nhất với toàn bộ dự án.

## 4. Tài liệu tham khảo
- GitHub Repository: `tigcom/bank-system`, `tigcom/MotorbikeSalesSystem`, `tigcom/YoutubeClone`.
